The Story Hugo Theme

Story is a beautiful Hugo theme. Hugo is a static website generator, which builds websites by rendering Markdown content into templates from themes like Story. The resulting site is fast, secure, and easy to build and maintain.

Story is a responsive theme that supports blogs, with many extra features such as presentations, math and music typesetting, and search. The Story design philosophy prioritizes simplicity and elegance, so it can be sophisticated and powerful at the same time.
